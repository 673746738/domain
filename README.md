# globalnav.ai - 全球AI导航页面

这是一个用于展示 globalnav.ai 域名价值的静态页面，包含市面上所有主流AI工具的列表。

## 功能特点

- 📱 响应式设计，支持移动端和桌面端
- 🎨 现代化的UI设计
- 🤖 涵盖100+主流AI工具，分为多个类别
- 📧 包含域名出售联系信息
- ⚡ 纯静态页面，加载速度快

## 部署方式

### 方式1: Vercel部署（推荐）

1. 将代码推送到GitHub仓库
2. 访问 [Vercel](https://vercel.com)
3. 导入GitHub仓库
4. 在Vercel项目设置中，将域名指向 `globalnav.ai`

### 方式2: Netlify部署

1. 将代码推送到GitHub仓库
2. 访问 [Netlify](https://netlify.com)
3. 导入GitHub仓库
4. 在Netlify项目设置中，添加自定义域名 `globalnav.ai`

### 方式3: GitHub Pages

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支作为源
4. 配置自定义域名 `globalnav.ai`
